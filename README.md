# ArtWala – Event Management System (Full Stack)

ArtWala is a full-stack event management web application that allows users to browse events and book tickets. It also includes an admin panel for managing events and bookings.

## Features
- Event browsing and ticket booking system
- Admin panel for event management (CRUD operations)
- User authentication with session handling
- Search and filtering of events using SQL queries
- Image upload and storage system
- Responsive user interface

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Live Demo
https://artwala.page.gd/index.php

## Key Highlights
- Full-stack CRUD application
- Relational database design
- Secure admin access using sessions
- Dynamic event handling system

## Purpose of Project
This project was developed to understand full-stack web development, database integration, and real-world application structure.
